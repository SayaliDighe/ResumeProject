# Resume Ranking AI Web App

A Django-based web application that analyzes resumes and matches them with job descriptions using NLP techniques.

## Features
- Upload resume (PDF / text)
- Upload job description
- NLP-based skill matching using TF-IDF
- Resume ranking score using cosine similarity
- Simple HR dashboard
- Django REST-ready architecture

## Tech Stack
- Django
- Python
- Scikit-learn
- NLP (TF-IDF)
- HTML, CSS
- SQLite

## Project Flow
1. User uploads resume and job description
2. Text is cleaned and processed
3. TF-IDF vectorization is applied
4. Cosine similarity calculates match score
5. Result is displayed to the user
